> **LUO OMNI** is the world's first **omni-language** - combining the full power of LUO + Python + JavaScript + SQL + Bash + AI into one unified programming system.

## What is LUO OMNI?

LUO OMNI is a revolutionary programming language that **breaks down the barriers** between all programming languages. It allows you to:

- 🌍 **Write in Plain English** - LUO's human-readable syntax
- 🐍 **Execute Full Python** - Complete Python runtime in browser
- 🟨 **Run JavaScript** - Native JavaScript execution
- 💙 **Query SQL** - Built-in SQLite database
- 🟣 **Execute Bash** - Shell command simulation
- ✨ **AI Integration** - Built-in AI assistance

**No company owns LUO.** It's decentralized and designed for both humans AND AI agents equally.

## Features

### 🌐 LUO - Plain English Programming
```luo
CREATE VARIABLE name SET "World"
PRINT "Hello, " + name
```

### 🐍 Full Python Support
```python
import math
from datetime import datetime

def calculate():
    return math.sqrt(16) * 2

print(f"Result: {calculate()}")
```

### 🟨 JavaScript Execution
```javascript
const users = [
  {name: "John", age: 30},
  {name: "Jane", age: 25}
];

users.forEach(u => console.log(u.name));
```

### 💙 SQL Database
```sql
CREATE TABLE users (
  id INTEGER PRIMARY KEY,
  name TEXT,
  email TEXT
);

INSERT INTO users (name, email) VALUES 
  ("John", "john@luo.dev"),
  ("Jane", "jane@luo.dev");

SELECT * FROM users;
```

## Quick Start

1. Open `index.html` in a modern browser
2. Wait for Python & SQL to initialize
3. Enter your Anthropic API key in the AI panel
4. Start coding!

## LUO OMNI Commands

### Plain English (LUO)
| Command | Description |
|---------|-------------|
| `PRINT "text"` | Output text |
| `CREATE VARIABLE x SET value` | Create variable |
| `SET x TO value` | Set variable |
| `ASK AI "question"` | Ask AI |

### Multi-Language Execution
| Command | Description |
|---------|-------------|
| `RUN PYTHON "code"` | Execute Python |
| `RUN JAVASCRIPT "code"` | Execute JavaScript |
| `QUERY "sql"` | Run SQL query |
| `BASH "command"` | Run Bash command |
